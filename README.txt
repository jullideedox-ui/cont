# Sign-in UI (HTML from Image)

This HTML recreates the provided screenshot as a clean, responsive page using only HTML & CSS (no external libraries).

## Files
- `index.html` — the page (self‑contained)

## How to use on GitHub Pages
1. Create a new public repo.
2. Upload `index.html` (drag & drop).
3. In **Settings → Pages**, select `Deploy from a branch`, choose `main` and `/ (root)`.
4. Save. Your page will be available at: `https://<username>.github.io/<repo>/`.
